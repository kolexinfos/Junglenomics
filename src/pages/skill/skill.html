THE SKILLMI QUIZ IS AN ADD ON FUNCTION OF JUNGLENOMICS GAMING SYSTEM THAT USES A PROPRIETARY ALGORITHM TO UNLOCK THE SPECIFIC SKILL SETS EMBEDDED IN EACH OF THE JUNGLENOMICS CARD TYPES. THE ALGORITHM IS ABLE TO

1. IDENTIFY EACH PLAYERS UNIQUE COGNATE TRIBE

2. IDENTIFY EACH PLAYERS CORE SKILL INTERESTS

3. IDENTIFY OTHER PLAYERS THAT BELONG TO HIS/HER TRIBE

4. IDENTIFY PLAYERS WITH COMPLIMENTARY AND OVERLAPPING SKILL SETS (USING THE CARD TAXONOMY FRAMEWORK AND CARD BASED ENTERPRISE ARCHITECTURE AS A GUIDE).
